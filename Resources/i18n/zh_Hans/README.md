## 安全警告

本项目及其相关开发者不对其造成的任何损失负责。

- 此程序包含闪烁画面，可能会造成不适。
- 此程序启用了扩展动态范围（EDR）渲染技术，并使用了最高亮度。
- 若要退出，请使用右键单击图标，并选择退出。
